# 📖 Overview

Wallpaper Engine is a full-stack web application designed for wallpaper enthusiasts. It provides a modern platform where users can browse thousands of wallpapers, create personalized profiles, save favorites, and unlock premium content through subscriptions.

The project focuses on providing a clean user experience while implementing secure authentication, payment integration, and efficient content management.

# 💻 Installation

## Clone Repository

```bash
git clone https://github.com/romitparmar/wallpaperEngine.git
```

---

## Move Project

Copy the project folder into:

```
xampp/htdocs/
```

---

## Start XAMPP

Start:

- Apache
- MySQL

---

## Import Database

1. Open phpMyAdmin
2. Create a database

```
wallpaper_engine
```

3. Import

```
database.sql
```

---

## Configure Database

Update your database credentials inside:

```php
includes/config.php
```

Example:

```php
$host = "localhost";
$user = "root";
$password = "";
$database = "wallpaper_engine";
```

---

## Run Project

Open your browser:

```
http://localhost/wallpaperEngine/
```
